英単語タイピングテスト（Node.js）

このプログラムは、コマンドライン上で動作する英単語タイピングテストです。
日本語訳と（オプションで）英単語の頭文字をもとに、対応する英単語をタイピングします。

───────────────────────
【必要な環境】
───────────────────────
- Node.js（https://nodejs.org/ からインストール可能）
- ターミナル（macOS/Linux）または コマンドプロンプト/PowerShell（Windows）

───────────────────────
【必要なファイル】
───────────────────────
- toeicTest.js（このプログラム）
- 英単語リスト（CSV形式の txt ファイル）
  形式：apple,りんご のように英単語と日本語訳をカンマ区切りで1行に記述

───────────────────────
【使い方：ターミナル操作】
───────────────────────
1. ターミナルまたはコマンドプロンプトを開く
2. typing_test.js が保存されているディレクトリに移動：

   ```bash
   cd /path/to/your/folder
   （例：cd ~/Documents/toeicTest）

3. 以下のようにコマンドを実行：

   node typing_test.js words1.txt
※ words1.txt は任意の英単語リストファイル名です
   
4. 表示される質問に従って y（はい）または n（いいえ）で答えるとテスト開始
(yと答えると英単語の頭文字がヒントとして表示されます。)

─────────────────────── 【テスト内容】 ───────────────────────

全20問出題

各問題では「日本語訳」と（必要に応じて）「頭文字」が提示されます

正しい英単語を入力してください

─────────────────────── 【操作方法】 ───────────────────────

正解：英単語を正確に入力 → +1点

不正解（1回目）：もう一度同じ単語が出題されます（再チャレンジ可）

不正解（2回目）：正解が表示され、次の問題へ

? を入力：例外的に即座に不正解となり、答えが表示されます

─────────────────────── 【スコア】 ───────────────────────

初期スコア：480点

正解1問ごとに +1点

「頭文字表示あり」の場合、テスト終了時に -1点

「頭文字表示あり」で全問正解で499

「頭文字表示なし」で全問正解で500です

全セクションで500を狙ってみましょう

─────────────────────── 【終了後】 ───────────────────────

最終スコアが表示されます

間違えた単語（英語 + 日本語訳）が一覧で表示されます

─────────────────────── 【備考】 ───────────────────────

同じ単語は2度出題されません

単語リストファイルは UTF-8 で保存してください

フォーマットは「英単語,日本語訳」です（例：banana,バナナ）
